
# 💼 Glassdoor Jobs Salary Prediction

> 🚀 Predicting salaries using Machine Learning & real-world job data from Glassdoor

---

## 📌 Project Overview

In today’s competitive job market, understanding salary trends is crucial for both job seekers and employers.
This project focuses on analyzing job listings from Glassdoor and building a **machine learning model to predict salaries** based on job attributes.

🔍 From raw, messy data → to clean insights → to accurate predictions.

---

## 🎯 Objectives

* Analyze salary trends across job roles, industries, and locations
* Perform **EDA with 15+ insightful visualizations**
* Engineer meaningful features from raw data
* Build and compare multiple ML models
* Select the best model for salary prediction
* Make the project **deployment-ready**

---

## 📊 Dataset Features

The dataset includes:

* 🧑‍💻 Job Title
* 🏢 Company Name
* 📍 Location
* ⭐ Company Rating
* 🏭 Industry & Sector
* 💰 Salary Estimate
* 🧾 Revenue & Company Size
